Terminology: About the words used in writing code

HTML

  • Element
    • A complete HTML command with several parts
      • opening tag
      • closing tag
      • sometimes contents
      • and sometimes attributes
  • Tag
    • basic command structure, surrounded by brackets to indicate what the command is applied to
  • Attribute
    • secondary description for action / structure of tag
    • example: horizonal rule (line) tag has several possible attributes that can be used: alighnment, size, color, shade/ noshade, length
    • insert labeled pictures of these things
  • Contents
    • what you want to apply the tag structure to
  • Closing and self closing
  • Note: heading and paragraph tags need a closing element. Other tags are self closing like line break (br), horizontal rule (hr), center and more
  • insert labeled pictures of these things
  • Comments can be left in the code, and they will not be rendered by the browser. Use "less than"bracket, exclamation point, dash, dash to open. Use dash, dash, "greater than" bracket to close

HTML Notes

Basic HTML Code Structure

  • head
    • information for browser on how to manage the page
    • background info not displayed on the website
    • includes:
      • declaration of document type as HTML
      • meta tags: information about the content for search engines.
      • meta tags can have attributes such as:
        • charset: declaration of unicode character set (always use UTF-8)
        • name: can have values of:
          • application name
          • author: name of site author
          • description: describes content/ intent of site. These are the descriptive words that show up under the page title in search engine results
          • generator
          • keywords: list of keywords for search engine, reflects topics covered by content of site
          • viewport
        • content: used as a descriptor with other attributes.
      • title of page. not shown on page but explains what it is to the internet
  • body
    • this section holds the code for everything that is displayed on the website
